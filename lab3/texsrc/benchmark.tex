\section*{Результаты}

Программа успешно реализует поставленную задачу. При запуске:
\begin{enumerate}
    \item Родительский процесс запрашивает имя файла для вывода результатов
    \item Затем запрашивает у пользователя числа для деления
    \item Передает данные в дочерний процесс через общую память
    \item Дочерний процесс выполняет вычисления и записывает результаты в файл
\end{enumerate}

Пример работы:
\begin{itemize}
    \item Пользователь вводит: \texttt{10 2 5}
    \item В файл записывается:
    \begin{verbatim}
    Child: Запущен. Файл вывода: output.txt
    10 / 2 = 5
    10 / 5 = 2
    ---
    \end{verbatim}
    \item При вводе: \texttt{10 0 5}
    \item В файл записывается:
    \begin{verbatim}
    Child: Запущен. Файл вывода: output.txt
    КРИТИЧЕСКАЯ ОШИБКА: Деление на ноль!
    Child: Получен SIGTERM. Завершение работы.
    \end{verbatim}
\end{itemize}
