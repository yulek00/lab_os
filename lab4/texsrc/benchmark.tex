\section*{Результаты}

Программа успешно создаёт четыре динамические библиотеки:
\begin{itemize}
    \item \texttt{libgcf\_euclid.so}
    \item \texttt{libgcf\_naive.so}
    \item \texttt{libsquare\_rect.so}
    \item \texttt{libsquare\_triangle.so}
\end{itemize}

Тестовая программа \texttt{prog\_static} корректно использует библиотеки, подключённые на этапе линковки.

Программа \texttt{prog\_dynamic} загружает библиотеки во время выполнения, поддерживает переключение реализаций по команде \texttt{3} и корректно вызывает функции через указатели.

Пример сессии:

\begin{verbatim}
Lab 4: Dynamic Loading
0 - Exit
1 <A> <B> - GCF
2 <A> <B> - Area
3 - Switch implementation
1 48 18
GCF Result: 6
3
Switched to: Naive & Triangle
2 4 5
Square Result: 10
0
Exiting...
\end{verbatim}

Все команды обрабатываются корректно. Ошибки загрузки библиотек обрабатываются с выводом диагностики.
