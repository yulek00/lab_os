\section*{Условие}

\subsection*{Цель работы}
Приобретение практических навыков диагностики работы программного обеспечения.

\subsection*{Задание}
При выполнении лабораторных работ по курсу ОС необходимо продемонстрировать ключевые системные вызовы, которые в них используются и то, что их использование соответствует варианту ЛР.
По итогам выполнения всех лабораторных работ отчет по данной ЛР должен содержать краткую сводку по исследованию написанных программ.

\subsection*{Средства диагностики}
strace

\section*{Метод решения}
Программы, реализованные для каждой лабораторной работы, запускаются с использованием strace. Им подаются тестовые входные данные, ожидается завершение их работы, после чего они корректно завершаются. Полученные логи анализируются для выявления ключевых системных вызовов, подтверждающих корректность реализации заявленных механизмов.
